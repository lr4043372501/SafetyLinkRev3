# SafetyLink — Marketing site UI kit

A high-fidelity recreation of the SafetyLink public site. The kit covers the homepage shell (nav, hero, feature blocks, stats band, customer story, pricing, footer) plus a clickable signup overlay.

## Files
- `index.html` — composes all sections into a scrollable homepage.
- `Nav.jsx`, `Hero.jsx`, `LogosBand.jsx`, `Features.jsx`, `Stats.jsx`, `Story.jsx`, `Pricing.jsx`, `CTA.jsx`, `Footer.jsx`, `SignupModal.jsx`.

## Notes
- Imports `../../colors_and_type.css` for tokens.
- Uses Lucide via CDN.
- Photography rendered via SVG placeholders in `assets/`.
