# SafetyLink — Web app UI kit

The compliance workspace: dashboard, audits / inspections list, inspection detail with checklist, and a profile/settings stub. Sidebar + topbar shell.

## Files
- `index.html` — interactive shell with view switching.
- `Shell.jsx` — sidebar + topbar.
- `Dashboard.jsx`, `Inspections.jsx`, `InspectionDetail.jsx`, `Training.jsx`, `Settings.jsx`.
