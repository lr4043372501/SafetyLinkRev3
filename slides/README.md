# SafetyLink slide templates

Six 1280×720 templates in the SafetyLink visual system:

- `title.html` — dark navy hero with chain-link motif and brand lockup.
- `section.html` — full-bleed dark section divider.
- `content.html` — three-column feature layout with icons.
- `big-number.html` — single dominant statistic with supporting copy.
- `comparison.html` — side-by-side before/after.
- `quote.html` — customer pull-quote.

Shared CSS in `styles.css`. All slides use Lucide icons via CDN and the SafetyLink logo PNG from `../assets/`.
